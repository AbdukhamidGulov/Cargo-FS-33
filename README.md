# 🚛 Cargo Logistics Bot

Telegram-бот для автоматизации логистических процессов. Помогает менеджерам и водителям управлять заявками и рассчитывать стоимость перевозок.

## 🚀 Функционал
* **Расчет стоимости:** Калькулятор доставки грузов.
* **Трекинг:** Статусы текущих заказов.
* **Администрирование:** Управление заявками через телеграм.
* **База данных:** Надежное хранение пользователей и заказов (MySQL).

## 🛠 Технологический стек
* **Язык:** Python 3.9
* **Фреймворк:** Aiogram (Асинхронный)
* **База данных:** MySQL + SQLAlchemy ORM
* **Логирование:** Logging (стандартная библиотека)

## ⚙️ Установка и запуск

1. **Клонируйте репозиторий:**
   git clone https://github.com/AbdukhamidGulov/Cargo-FS-33.git

2. **Установите зависимости:**
   pip install -r requirements.txt

3. **Настройте переменные окружения:**
   Переименуйте файл `.env.example` в `.env` и заполните его своими данными.

4. **Запустите бота:**
   python main.py
