# 🚛 Cargo Logistics Bot

Telegram-бот для автоматизации логистических процессов и управления грузоперевозками. Разработан для оптимизации взаимодействия между менеджерами, водителями и клиентами.

## 🚀 Основной функционал

* **Расчет стоимости:** Калькулятор доставки грузов с учетом веса, объема и типа транспорта.
* **Трекинг грузов:** Отслеживание статуса отправлений по ID.
* **Админ-панель:** Удобный интерфейс для менеджеров (добавление заявок, смена статусов).
* **Уведомления:** Автоматическая рассылка статусов клиентам.
* **База данных:** Хранение истории заказов и клиентской базы.

## 🛠 Технологический стек

Проект выполнен с использованием современного стека технологий для обеспечения надежности и скорости работы:

* **Язык:** Python 3.10+
* **Фреймворк:** Aiogram (Асинхронная работа)
* **База данных:** PostgreSQL / SQLite (через ORM)
* **Деплой:** Docker & Docker Compose
* **Логирование:** Loguru

## ⚙️ Установка и запуск

1. **Клонируйте репозиторий:**
   git clone https://github.com/AbdukhamidGulov/Cargo-FS-33.git

2. **Создайте виртуальное окружение:**
   python -m venv venv
   source venv/bin/activate  # для Linux/MacOS
   venv\Scripts\activate     # для Windows

3. **Установите зависимости:**
   pip install -r requirements.txt

4. **Настройте переменные окружения:**
   Создайте файл .env и укажите ваш BOT_TOKEN и доступы к БД.

5. **Запустите бота:**
   python app.py
